# Hotel-Revenue-Optimization-Analytics

## Objective
This project analyzes hotel revenue performance using Power BI to identify pricing, occupancy, and demand optimization opportunities in the hotels.

## Business Problem
The revenue team needed visibility into property-level performance and clarity on which hotels, cities, and booking channels were driving revenue gaps.

## Tools Used
- Power BI
- Power Query
- DAX

## Key Metrics
- RevPAR
- ADR
- Occupancy %
- DSRN, URN, BRN
- Realization
- Weekday vs Weekend performance
- Channel-level bookings

## Approach
- Cleaned and transformed booking data using Power Query
- Built a star-schema data model
- Created DAX measures for capacity, utilization, and trends
- Designed an executive Power BI dashboard
- Generated actionable revenue and pricing insights

## Key Insights
- Identified low ADR properties with high occupancy potential
- Highlighted pricing sensitivity between weekdays and weekends
- Flagged underperforming hotels driving city-level revenue decline
- Suggested promotion strategies without disrupting OTA rankings
